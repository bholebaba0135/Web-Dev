Courses Offered in SCOPE


  • Data Structures and Algorithms

    The course comprises of Binary Search Trees, Stack,Linked Lists,Hashing,Tower of Hanoi,Djikstra's Algorithm

  • Internet and Web Programming

    The course teaches students how to create interactive and responsive websites using HTML,CSS,Js,Ajax

  • Java Programming

    The course comprises of basic java, DSA, OOPS concept, JDBC,JavaFX,Exception Handling,MultiThreading and Java Servlets

  • Operating Systems

    The course shall entail virtualisation, Scanning Algorithms, Process Synchronization Algorithms and Inter Process Communication

  • Database Management Systems

    In this course, students shall learn ER diagrams,Normalizations,Heuristic Query Optimization and Relational Algebra

  • Machine Learning

    In this course, students shall learn numpy, matplotlib, pandas and geopandas libraries along with concepts like markov chains
